# CodifyX

This repository contains the CodifyX marketplace platform with separate web apps, APIs, shared modules, tests, docs, and deployment assets.

## Structure

- apps/: frontend and backend application entrypoints
- shared/: reusable domain modules
- tests/: automated test suites grouped by area
- docs/: architecture, API, security, legal, and deployment guides
- docker/: container assets
- backups/: archived artifacts
